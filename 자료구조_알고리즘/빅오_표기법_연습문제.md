- ```java
  //2.
  int a = 0;
  for (i = 0; i < N; i++) {  //N
      for (j = N; j > i; j--) { // N, N-1, N-2,.... 1 
          a = a + i + j;
      }
  }
  ```
  - N + (N–1) + (N–2) + ... 1 = N(N+1)/2 
  - 시간복잡도: O(N^2)
- ```java
  //3.
  int i, j, k = 0;
  for (i = n / 2; i <= n; i++) {  // n/2
      for (j = 2; j <= n; j = j * 2) { //log2(n)
          k = k + n / 2;
      }
  }
  ```
- 2, 4, 8, ... 2^k <=2^n => k <= log2(n)
- 시간복잡도: O(nlogn)
